# Social Suite

Social Suite ("Krowdr") is a demo social media marketing tool UI built with Next.js App Router, Tailwind, and Radix.

## Local dev

1. Install deps: `npm install`
2. Run: `npm run dev`
3. Open: `http://localhost:3000`

## Demo access

- Go to `/login`
- Demo credentials:
  - Email: `demo@krowdr.com`
  - Password: `krowdr-demo`

The app uses a demo cookie session to protect dashboard routes (`/overview`, `/schedule`, etc.).

## Scripts

- `npm run dev` - start dev server
- `npm run lint` - lint
- `npm run typecheck` - TypeScript check
- `npm run build` - production build
